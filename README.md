# KalyanaOne

KalyanaOne is a community-based matrimony web app focused on a simple, trusted, and user-first matchmaking experience. The app includes a public landing page, onboarding, authentication, profile creation, search, messaging, and premium plans.

## Highlights

- Public landing page with plan comparison and early access messaging
- Auth flows for signup, login, and password reset
- Profile setup, preferences, and privacy controls
- Match discovery, shortlisted profiles, and insights
- In-app messaging and notifications
- Admin verification route for profile reviews

## Tech Stack

- React 19 + Vite
- React Router
- Firebase
- MUI, Emotion, React Icons, Lottie

## Project Structure

- src/components: feature-based UI modules (landing, auth, home, messages, settings, premium)
- src/context: global app and auth context
- src/clientFunctions: client helpers
- src/models: reusable UI sheets and popups

## Scripts

- dev: start Vite dev server
- build: production build
- lint: run ESLint
- preview: preview production build

## Handoff Notes

- Install dependencies: npm install
- Start dev server: npm run dev
- Production build: npm run build
- Preview build: npm run preview
